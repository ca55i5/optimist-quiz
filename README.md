# Optimistディンギー クイズ

Optimistディンギーの帆走ルール・旗・ホイッスル・セッティングなどを楽しく学べるクイズアプリです。

## アクセス

GitHub Pagesで公開中：https://[ユーザー名].github.io/optimist-quiz/

## 内容

全70問、10カテゴリ：

- ⚖️ ルール（RRS） 8問
- 🚩 旗（テキスト） 13問
- 👀 旗を見る（ビジュアル） 12問
- 📣 ホイッスル 6問
- 🔧 セッティング基本 8問
- 📐 セッティング詳細 10問
- ⛵ 艤装・パーツ 6問
- 💨 風・操船 6問
- 🦺 安全 4問
- 📚 クラス知識 5問

## 特徴

- 漢字すべてにふりがな付き（子どもでも読める）
- カテゴリ別練習・全問シャッフル・ランダム10問の3モード
- 旗の絵をSVGで表示（実物を覚えられる）
- 解答ごとに解説付き

## ランキングを他端末で共有する設定

このアプリはデフォルトではブラウザの localStorage に保存します。
他端末共有を有効にするには Supabase を設定してください。

1. Supabase プロジェクトを作成
2. SQL Editor で次を実行してテーブル作成

```sql
create table if not exists public.quiz_rankings (
	id bigint generated by default as identity primary key,
	mode text not null,
	bucket text not null,
	score int not null,
	total int not null,
	pct numeric(5,1) not null,
	name text not null,
	played_at timestamptz not null default now()
);

create index if not exists idx_quiz_rankings_bucket on public.quiz_rankings(bucket);
create index if not exists idx_quiz_rankings_sort on public.quiz_rankings(pct desc, score desc, played_at desc);
```

3. RLS（Row Level Security）を有効化して、読み書きポリシーを追加

```sql
alter table public.quiz_rankings enable row level security;

create policy "public read rankings"
on public.quiz_rankings
for select
to anon
using (true);

create policy "public insert rankings"
on public.quiz_rankings
for insert
to anon
with check (true);
```

4. `config.js` に Supabase 情報を設定

```js
window.OPTIMIST_QUIZ_CONFIG = {
	supabaseUrl: 'https://YOUR_PROJECT_ID.supabase.co',
	supabaseAnonKey: 'YOUR_SUPABASE_ANON_KEY'
};
```

設定後は Supabase にランキングが保存され、別端末からも同じランキングを参照できます。
未設定時や通信失敗時は自動的に localStorage 保存へフォールバックします。

既存の localStorage ランキングがある場合は、クラウド設定が有効になった初回アクセス時に自動で Supabase へ移行します。
全件の移行成功後に localStorage 側のランキングは自動削除されます（失敗時は削除されません）。
